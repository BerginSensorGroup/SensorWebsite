Appendix B: Arduino and Teensyduino Installaction for PC

To install Arduino and Teensyduino: Note that the latest version of Arduino has compatibility issues with Teensy, therefore we use a slightly older version that can be found here
I use 1.6.8.

  1. Execute arduino-1.6.8-windows.exe
  2. If prompted to allow this app to make change to your computer, choose “Yes”
  3. Agree to the license agreement
  4. Leave everything as default, “Next >” – “Install”
  5. Wait for the installation to complete, choose “Close” when done

Teensy 1.28 Install:

  1. Execute TeensyduinoInstall.exe
  2. Follow the prompts to install this version of Teensyduino. This will need to be installed in the same location as the Arduino software. On the Select Arduino Folder window, choose the path: C:/Program Files(x86)/Arduino/, and click “Next”.

To install libraries: make sure all folders included in the screenshots are present in the designated location. Libraries are located in two places on your computer - C:/ Program Files (x86)/Arduino/ libraries, and Your PC:/ Documents/ Arduino / libraries

The SD folder is very important and is normally the one that most computers miss. If the SD folder is not present, download it here
