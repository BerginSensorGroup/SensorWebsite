Appendix A: Arduino and Teensyduino Installaction for Mac #back to top

  1. Make sure any already present versions of Arduino have been uninstalled.
  2. Download the Arduino version 1.6.8 Installer from this link: Mac OS-X Arduino Installer
  3. Install Arduino; make sure that the version is 1.6.8.
  4. Run and then close Arduino.
  5. Download the Arduino Folders from this link: Arduino Folders
  6. Go to your Documents folder, delete the folder called “Arduino”, and replace it with the downloaded folder called “Arduino (Replaces Arduino in Documents)”. Rename this folder “Arduino”.
  7. Place the downloaded folder called “Arduino (Place Arduino Application Inside)” wherever you want to store the Arduino application (it doesn’t have to be in the Applications folder). Place the Arduino application into the downloaded folder, and rename the folder whatever you want.
  8. Download the Teensyduino version 1.28 Installer from this link: Mac OS-X Teensyduino Installer
  9. Run the Installer and follow the instructions; make sure the version is 1.28; when it prompts you to select an install location, navigate to the Arduino application and open that; when it asks you which additional libraries to install, install all of them.
  10. Open either the 2016 or the 2017 versions of the code in Arduino, click “Tools”, and set the board type to Teensy 3.2 / 3.1. Click the Verify button. The code should successfully compile.
