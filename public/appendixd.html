Appendix D: Additional Files You May Need #back to top

The following files should have been provided to you along with this document and/or may be necessary to complete the steps outlined here:

  • Gerber files to send in to PCB manufacturer
  • Bill of materials (excel document) to send to PCB manufacturer if ordering PCB with parts soldered on
  • Code to upload to the teensy (a .ino file)
  • 3 template files to upload to the SD card (CALI.txt, LAB.txt, TEENSY.txt)
  • Folders containing the R plotter and calibrator files for running the apps (the code files), and empty named files for housing the versions of the files as the code runs
  • Teensy libraries (can also be found online)
