Appendix D: Additional Files You May Need

The following files should have been provided to you along with this document and/or may be necessary to complete the steps outlined here:

  1. Gerber files to send in to PCB manufacturer
  2. Bill of materials (excel document) to send to PCB manufacturer if ordering PCB with parts soldered on
  3. Code to upload to the teensy (a .ino file) and Teensy Libraries (can also be found online)
  4. Template files to upload to the SD card (CALI.txt, LAB.txt, TEENSY.txt)
  5. Folders containing the R plotter and calibrator files for running the apps (the code files), and empty named files for housing the versions of the files as the code runs
